### 1. Wave Array

[**GFG**](https://www.geeksforgeeks.org/problems/wave-array-1587115621/1?track=amazon-arrays&batchId=192)


_If array is sorted:_
```java
class Solution {
    public void sortInWave(int arr[]) {
        // code here
        for(int i=0; i<arr.length; i+=2) {
            if(i+1<arr.length) {
                int temp = arr[i];
                arr[i] = arr[i+1];
                arr[i+1] = temp;
            }
        }
    }
}
```
_If array is not sorted:_
```java
class Solution {
    public void sortInWave(int arr[]) {
        int n = arr.length;

        for (int i = 0; i < n; i++) {
            // For even index, arr[i] should be greater or equal
            if (i % 2 == 0) {
                if (i > 0 && arr[i] < arr[i - 1]) {
                    swap(arr, i, i - 1);
                }
                if (i < n - 1 && arr[i] < arr[i + 1]) {
                    swap(arr, i, i + 1);
                }
            } 
            // For odd index, arr[i] should be smaller or equal
            else {
                if (i > 0 && arr[i] > arr[i - 1]) {
                    swap(arr, i, i - 1);
                }
                if (i < n - 1 && arr[i] > arr[i + 1]) {
                    swap(arr, i, i + 1);
                }
            }
        }
    }

    private void swap(int[] arr, int i, int j) {
        int temp = arr[i];
        arr[i] = arr[j];
        arr[j] = temp;
    }
}
```

### 2. Equilibrium Point
[**GFG**](https://www.geeksforgeeks.org/problems/equilibrium-point-1587115620/1)

```java
class Solution {
    // Function to find equilibrium point in the array.
    public static int findEquilibrium(int arr[]) {
        // code here
        int n = arr.length;
        
        int totalSum = Arrays.stream(arr).sum();
        
        int leftSum = 0;
        
        for(int i=0; i<n; i++) {
            int rightSum = totalSum - leftSum - arr[i];
            if(leftSum == rightSum) {
                return i;
            }
            leftSum += arr[i];
        }
        return -1;
    }
}
```

### 3. Missing Number
[**GFG**](https://www.geeksforgeeks.org/problems/missing-number-in-array1416/1)
```java
class Solution {
    int missingNum(int arr[]) {
        // code here
        int n = arr.length+1;
        long totalSum = 0;
        for(int x: arr) {
            totalSum += x;
        }
        long nSum = (long)n*(n+1)/2;
        long ans = nSum - totalSum;
        return (int) ans;
    }
}
```
_Using Bit Manipulation:_
```java
class Solution {
    int missingNum(int arr[]) {
        // code here
        int n = arr.length+1;
        int xor = 0;
        
        for(int i=1; i<=n; i++) {
            xor ^= i;
        }
        
        for(int x: arr) {
            xor ^= x;
        }
        
        return xor;
    }
} 
```
_Java one liner:_
```java
class Solution {
    int missingNum(int[] arr) {

    }
}
```
